## 项目介绍:

`Arterial` (动脉)项目,功能如下:

支持域名或IP地址为服务地址发送文件。

## 使用

```bash
Usage:
   Server: -s -f [File name] -p [local port]
   Client: -c -f [File name] -h [remote host] -p [remote port]
```
